# Customer Satisfaction Prediction using Machine Learning

## Project Overview

This project analyzes customer support data and predicts Customer Satisfaction (CSAT) scores using Machine Learning models.

## Dataset

The dataset contains customer support interactions including:

* Support Channel
* Category
* Sub-category
* Agent Details
* Resolution Time
* Agent Shift
* Agent Tenure
* CSAT Score

## Data Preprocessing

* Missing value handling
* Feature engineering
* Label Encoding
* Resolution time calculation
* Date-time feature extraction

## Exploratory Data Analysis

Visualizations include:

* CSAT Distribution
* Support Channel Distribution
* Category Distribution
* Agent Shift Analysis
* Agent Tenure Analysis
* Resolution Time Analysis
* Model Comparison

## Machine Learning Models

1. Random Forest Classifier
2. XGBoost Classifier
3. Extra Trees Classifier

## Results

The models were evaluated using:

* Accuracy
* Precision
* Recall
* F1 Score

* Python
* Pandas
